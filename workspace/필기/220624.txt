● windowBuilder  사용 방법
파일에서 마우스 우클릭 - open with - window builder - console창 뜨는 쪽에 design 누르면 확인가능
▶ jar 저장하기
파일 폴더 우클릭 export - java - Runnable java file - 파일 찾아서 선택 - browse에서 경로 설정 후 저장(파일형태 jar로 변경해주기)
▶ properties에서 화살표 누르면 속성 확인 가능

●JFrame
▶ JCheckBox()
: 추상적인 버튼을 상속받는 클래스
- 생성자에 체크박스명 설정 가능
- setSelected() : 불리언으로 표현, 선택여부
- isSelected() : 선택되었는지? 불리언값 출력
▷ getStateChange(); : 상태 변화 알려주는 메소드 -> int값 반환
▷ SELECTED : 선택됨

▶ JRadioButton()
- ButtonGroup과 짝으로 온다고 생각하면 됨
- ButtonGroup으로 묶어주면 버튼 묶는 그룹 생성 -> 하나를 선택하면 나머지가 선택 해제, 중복이 되지 않고 하나만 선택될  수 있도록 함

▶ addItemListener : 아이템리스너를 전달받아서 실행하는 메소드
