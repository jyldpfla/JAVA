<어제 복습>
기초형은 각자의 크기를 가지고 있고 변수 안에 값이 들어가 있다
참조형은 실제 값은 박스에 들어있지만, 다른 곳에 들어있는 정보를 찾을 수 있게 화살표로 연결(위치를 앎)되어 있다.

*cha boolean int double은 꼭 기억*

논리형 boolean, 문자형 character - && ||은 논리형을 비교해주는것이므로 true,false를 결과값으로 가지는 것에 붙여주면 됨

정수형 - 박스의 크기가 달라서 그에 맞는 박스 필요, 넉넉한 크기에 맞춰서 데이터 타입 사용해야 함
-비트 수를 꼭 알아야하지 않음
- 기본적으로 int로 표시되므로 long 쓰고 싶으면 뒤에 단위 L 써주면 됨

실수형 - 정확한 값을 알고 싶을 때, 소수점 자리까지 알고 싶을 때 실수형 사용
- double이 2배 정도 정확하므로 double 사용 추천
- 기본적으로 double로 인식되므로 float는 단위 F
- 정수 + 실수 = 실수타입으로 출력

printf : 서식이 있는 것 출력
- %f : 소수점 자리 나타내기
- %d : 정수 값
- %s : 문자열

문자형('')
: ""문자열과는 차이, 기초형 데이터 타입이라 char사용, 글자 딱 하나만 사용(한글도 가능)
-\n : 두 글자로 이루어졌지만 하나의 문자로 취급, 개행 문자, 줄바꿈 가능
-문자열은 문자를 여러가지 가질 수 있음
-\t : 띄어쓰기를 할 수 있는 문자, 정해진 칸 수만큼의 공백으로 들여쓰기 (char tab = '\t';)
-\b : 백스페이스, \f : 폼피드 등
- ' " \들을 하나의 문자로 취급하고 싶을 때 앞에 \ 붙여주면 됨, \는 ""안에 \\ 두 번 써주면 하나의 \가 나옴

boolean : 결과값으로 true, false만 출력되는 타입

대입연산자
변수값을 교환하고 싶을 때, 값보관을 위해 새로운 공간을 만들어 값을 저장하고 값을 꺼내 씀
-복합대입연산자: 산술연산자+대입연산자 (a *= 2;)
-증감연산자 : 변수의 앞뒤에 1만큼 더하고 빼도록 하는 연산자
	-주로 뒤에 많이 붙여 사용
-관계연산자 : 두 개의 값을 비교해서 true, false를 값으로 줌
-논리연산자 : and(&&)둘 모두 true 여야 true, or(||) 둘 중 하나라도 true면 true
	-각각의 관계연산을 논리연산자로 합쳐서 하나의 결과값으로 출력 가능
