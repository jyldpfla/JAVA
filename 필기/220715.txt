● JDBC를 이용해 연결하고 사용하기
1. 테이블 만들기
- ex. String query = "CREATE TABLE A (B VARCHAR(10) PRIMARY KEY)"; // 대소문자 구분X, 테이블 생성
2. 명령어 수행해줄 객체 만들기
▶ statement 객체 : 명령어를 수행해주는 객체
- connection객체에서 얻어올 수 있음
- 사용하고 닫아주어야함
▶ 메소드  ① excuteUpdate() : 결과를 int로 반환 받을 수 있는 경우에 사용하는 메소드수정작업(추가, 생성, 변경, 삭제)을 동작하고 	싶을 때
	-ex. insert into 결과 => '1'줄로 표현 되었다
	② executequery() : 테이블의 형태로 결과를 받아볼 수 있는 객체인 resultset을 반환할 수 있도록 한 메소드
	- ex select from 결과 => 테이블로 결과 출력
▶ resultset : select문의 결과값으로 resultset을 반환하는 객체
- 사용 후 닫아주어야함

● 테이블 
- 무결성 법칙 : 각 레코드(행)은 중복되지 않아야 함
