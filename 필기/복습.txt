● 필드
- , : 같은 타입인 여러 필드 한번에 선언 시 구분자
● 매개변수 (= parameter) :  메소드 수행에 필요한 입력값을 저장하는 변수 
● 생성자
: 초기화하기 위해 만들어짐
- 생성자를 따로 생성해주지 않으면, 파라미터를 아무것도 가지지 않는 빈 생성자가 자동으로 생성됨 -> 참조를 필요로 하지 않음
● getter와 setter
: Private 으로 선언된 인스턴스 필드에 접근하기 위해서 Getter와 Setter를 사용
1) setter :  Main 함수에서 인스턴스 필드에 접근해서 사용하는 함수입니다. 매개변수로 멤버 변수를 초기화
2) getter : Main 함수에서 인스턴스 필드에 접근해서 사용하는 함수입니다. 초기화된 멤버 변수를 return리턴 합니다.(반환 값을 가지게 됨)
멤버변수가 어떤 값으로도 초기화되지 않았다면 null을 리턴
● Scanner : 사용자의 입력을 받기 위한 클래스
- 사용을 위해 java.util클래스에 있는 Scanner 클래스를 import한 뒤, 클래스 객체 생성(인스턴스화)를 해주어야함
● while문 : 
- while(true,false값을 결과로 가지는 조건) {조건이 true일 때 실행되는 실행문}
▶ do-while문 - 조건 결과에 상관없이 무조건 1번은 실행됨
