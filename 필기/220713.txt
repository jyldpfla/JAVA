● MySQL 명령어
1. create database db이름; : db 생성
2. create table 테이블이름(컬럼명 컬럼타입, 컬럼명 컬럼타입); : 테이블 만들기
- sql은 문자열타입은 길이도 지정해주어야함
- varchar() : 문자열을 나타내는 명령어, ()는 길이
- int : 정수
3. insert into 테이블이름 (넣고자하는열의이름, 넣고자하는열의이름) values (넣고자하는정보, 정보(열순서대로)) : 하나의 행을 넣기
- 앞의 열 순서대로만 행 정보 추가해주면 값 제대로 들어감
4. insert into 테이블이름 (넣고자하는열의이름, 넣고자하는열의이름) values (넣고자하는정보, 정보(열순서대로)), (넣고자하는정보, 정보(열순서대로)) : 여러줄의 행 넣기
5. delete from 테이블명 : 테이블의 모든 행 삭제
- delete from 테이블명 where 조건 : 조건에 맞는 행 삭제
6. drop table 테이블명 : 테이블 삭제
   drop database db이름 : db 삭제
7. rtrim(문자열) : 우측 공백 제거/ ltrim(문자열) : 좌측 공백 제거
8. 열이름 NOT NULL : NULL값 허용 X
9. 열이름 UNIQUE : 중복X
10. ALTER TABLE 테이블명 하고자하는동작 : 테이블 구조 변경
- ADD COLUMN : 컬럼 추가
11. UPDATE 테이블명 SET 컬럼명 = 새값 where 조건 : 바꾸려고하는 부분 변경
12. ADD CONSTRAING FOREIGN KEY (참조받을 키이름) REFERENCES 테이블명 (참조할 키 이름) : 외래키 만들기
13. COUNT(컬럼명) : 개수
- 보통 * 사용, NULL값을 가지는 컬럼의 경우 다르게 동작하므로(NULL제외)
14. SUM(컬럼명) : 합
15. AVG(컬럼명) : 평균
16. MAX(컬럼명)/MIN(컬럼명) : 최대/ 최소
● 테이블
- 테이블명은 고유해야함
▶ 열(column) : 이름을 가지고, 특정한 값이 들어감

▷ sql에서 문자를 나타내는 type
① char(문자열길이) : 고정된 길이값을 가짐 - 문자열길이에 맞춰지도록 값이 작아도 공백(빈칸)으로 그 값을 채워서 문자열길이를 맞춤
② varchar(문자열길이) : 가변 길이값을 가짐 - 문자열길이보다 모자라도, 그에 맞게 길이값이 조정됨

▶int는 소수점 표현이 불가하므로, int형에 실수 넣으면 반올림 실행, double형에 int형 넣으면 실수화됨
- 숫자 표현 : long, bigint 등도 존재
▶ double : 부동소수점 이용 -> 근사값 표현

▷ 중복되지 않는 KEY를 통해 고유한 자료 하나만 가리키기
1. PRIMARY KEY (PK) : 주요키, NULL값을 허용하지 않음(무조건 값이 있어야 함), 같은 값을 가지는 pk존재 불가(고유, 중복되지 x)
=> NOT NULL, UNIQUE
- 기본적으로 테이블 만들 때는 키 하나씩 존재해야 함
- 문자형도 PK 가능
- auto_increment : 값이 하나씩 자동으로 늘어남, key값 하나씩 증가해서 따로 부여하지 않아도 됨(감소하지 X, 앞의 값 사라져도 들어가지 않음), 값 추가 시도해서 실패해도 값은 계속 오름(사용한 키라고 생각해서 버림)

▷ KEY값을 이용해 테이블 간 연결하기 -> 구조 변경
▷ 외래키 : 다른 테이블의 키를 참조하는 키

▷ ERD : 테이블간의 관계를나타내주는 그림
- database - reverse engineer - next 

▷ 서브 뭐리 : 쿼리 안의 쿼리, 값을 구할 때 사용
- FROM문, WHERE IN, COLUMN명 안에서도 사용 가능

▷ 집계 함수
COUNT() / SUM() / AVG()
