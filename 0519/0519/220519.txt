● printf : 서식이 있는 문자 출력, 줄 바꿈은 안됨
-%d : 정수 하나가 올 수 있는 서식 문자, printf에서만 사용
-기록예시 : System.out.printf("Hello %dWorld\n, 1); : %d에 올 숫자 ,뒤에 쓰면 됨
-\n : 개행문자, 줄바꿈을 하게 해줌

char : 문자를 나타내는 변수 지정

● 초기화 : 변수를 사용할 때 가장 먼저 주어지는 값을 줄 때

Boolean은 참(True) 혹은 거짓(False) 중 하나의 값을 출력하는 데이터 타입이며 bool이라고 부르기도 한다. 비교(관계) 연산자는 비교 결과를 참(True) 혹은 거짓(False) 중 하나로 나타내 주는 기호

● 논리연산자
and (&&) : 둘다 참일 때 true, 둘 중에 하나라도 거짓이면 false
or (||) : 둘다 거짓이면 false, 둘 중에 하나라도 참이면 true
